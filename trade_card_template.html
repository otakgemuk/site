📖 How to use this template

  1. Click any text to edit it — dates, names, numbers, everything is editable
  2. Card color / result: Find class="card" in the code and change to: class="card win"  |  class="card loss"  |  class="card be"
  3. Direction: Change class="dir short" to class="dir long" and swap text to LONG ▲
  4. P&L color: Change class="stat-value" to class="stat-value sv-green" (or sv-red / sv-gold)
  5. Result color: Change class="stat-value" to class="stat-value sv-green">WIN ✓ or sv-red">LOSS ✗ or sv-gold">B/E —
  6. Add a trade: Copy an entire <div class="card">...</div> block and paste it inside <div class="cards-grid">
  7. Add a tag: Copy a <span class="tag"> line. Use class="tag hl" for gold highlight
  8. Remove a trade: Delete its <div class="card">...</div> block
Delete this entire <div class="howto"> block when you're done reading
